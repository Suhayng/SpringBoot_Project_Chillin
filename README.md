## ✔️ 기획의도
- 학부모들이 경험과 의견을 자유롭게 공유하며 서로 도움을 주고 받는 유익한 연결 형성
- 실용적인 교육 자료를 제공하여 자녀의 성장과 교육에 도움이되는 정보를 얻을 수 있도록 함
- 유용한 게시글은 추천하고, 부적절한 게시글은 신고를 하며 커뮤니티의 질서 유지
